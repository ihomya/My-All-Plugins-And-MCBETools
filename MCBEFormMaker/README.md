## 詳細
- 開発言語 Java  
- 実行可能環境 Java8以上  
- 実行方式 jar
- 開発環境 Windows10 HomeEdition / Eclipse Neon.2 Release (4.6.2)  
- ライセンス GNU Public License (GPL) version 3.0  
  
## 主な機能
- MCBE フォーム開発（alpha1ではSlider並びにStepSliderはサポートされていません。）  
- フォームをjsonとしてエクスポート（実際に使えます）  
- レスポンス確認機能  
他はソフト内のヘルプからご確認ください。
